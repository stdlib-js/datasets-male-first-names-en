
{{alias}}()
    Returns a list of common male first names in English speaking countries.

    Returns
    -------
    out: Array<string>
        List of common male first names.

    Examples
    --------
    > var list = {{alias}}()
    [ 'Aaron', 'Ab', 'Abba', 'Abbe', ... ]
